## 📚 Projects Overview

### 🕰️ Bharat Clock *(Root Project)*
A real-time clock application styled with **Bootstrap**. Demonstrates component composition with `ClockHeading`, `ClockSlogan`, and `CurrentTime` components.

### ⚛️ React Starter
A minimal React + Vite boilerplate to kickstart new experiments quickly.

### 🍎 Food Items — Fragments & Lists
Explores **props**, **lists rendering** with `.map()`, **conditional rendering**, and Bootstrap's list-group components.

### 🎨 Bootstrap Practice
Hands-on practice integrating **Bootstrap 5** button variants inside a React + Vite project.

### 🧮 Calculator v1
A calculator UI built with **CSS Modules**, featuring a `Display` component and a dynamically rendered `ButtonsContainer` using `.map()`.

### ✅ Todo App v1 & v2
Full CRUD todo application using **useState** hook — supports adding new todos with name & date, deleting todos, and conditional empty-state rendering.

## ⚡ Quick Start

```bash
# Clone the repository
git clone https://github.com/mananjani2102/react-practice.git
cd react-practice

# Install dependencies
npm install

# Start development server
npm run dev
```

> **Note:** Each sub-project inside `projects/` has its own `package.json`. Navigate into the project folder and run `npm install` separately.
